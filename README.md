# 👨‍🏫 examples

A collection of example projects for learning and reference, organized by language and topic.

## 📂 Structure

### 📖 [Kotlin](kotlin/)

Android applications built with Kotlin, Jetpack Compose and Material Design 3.

| Project | Description |
|---|---|
| [crud-room-jetpack-compose](kotlin/crud-room-jetpack-compose/) | Notes app with CRUD using Room database |
| [deeplinks-jetpack-compose](kotlin/deeplinks-jetpack-compose/) | App and web deeplinks example |
| [dialogs-jetpack-material-3](kotlin/dialogs-jetpack-material-3/) | Material Design 3 dialog component |
| [hello-world-jetpack-material-3](kotlin/hello-world-jetpack-material-3/) | Hello world with reactive text field |
| [navigation-jetpack-compose](kotlin/navigation-jetpack-compose/) | Bottom navigation bar example |
| [machine-learning/dogs-vs-cats](kotlin/machine-learning/dogs-vs-cats/) | Image classifier (cat vs dog) using TFLite |
