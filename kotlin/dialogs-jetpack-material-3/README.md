# ⭐ ‘Dialog’ component with Kotlin + Jetpack Compose + Material Design 3

![‘Dialog’ component with Kotlin + Jetpack Compose + Material Design 3](https://github.com/dacape-dev/examples/blob/main/kotlin/dialogs-jetpack-material-3/dialog.png#center)

`🤖 Android` `📖 Kotlin` `🚀 Jetpack Compose` `🖌️ Material Design 3`

Simple application with an example of the material design dialog component.

## 📚 Dependencies

* compose_version = ‘1.1.1’
* material3_version = ‘material3:1.0.0-alpha02’
* lifecycle-viewmodel-compose = ‘2.6.1’

```
    implementation 'androidx.core:core-ktx:1.7.0'
    implementation 'androidx.lifecycle:lifecycle-runtime-ktx:2.3.1'
    implementation 'androidx.activity:activity-compose:1.3.1'
    implementation "androidx.compose.ui:ui:$compose_version"
    implementation "androidx.compose.ui:ui-tooling-preview:$compose_version"

    implementation 'androidx.compose.material3:material3:1.0.0-alpha02'
    implementation 'androidx.lifecycle:lifecycle-viewmodel-compose:2.6.1'
```
