# 🐍 Python

## 🤖 Machine Learning

### 🚀 How to run

#### Google Colab (recommended)

1. Open [Google Colab](https://colab.research.google.com/).
2. Upload the `.py` script or paste its contents into a new notebook cell.
3. Run the cells in order.
4. Download the output files (`.tflite`, `labels.txt`) when prompted.

#### Local environment

```bash
pip install tensorflow tensorflow-datasets
python script.py
```

> **Note:** GPU recommended for training. Use a virtual environment to isolate dependencies.
